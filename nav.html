📅 12日深度奢華行程
D1ARRIVE第 01 天
ウルムチ到着 · 接機

出發地 ✈️ 烏魯木齊 (24H 接機)

全天
抵達烏魯木齊，司機專車接送至酒店。自由活動推薦前往【自治區博物館】（需提前一天預約）或【二道橋國際大巴扎】。
🏨 住宿：新旅逸格酒店 / 桔子水晶酒店或同級【四鑽】
🕌 西域大巴扎 · 體驗熱情民族風情
D2TRAVEL第 02 天
天山を越える砂漠の海

烏魯木齊 ➔ 烏尉高速 ➔ 天山勝利隧道 ➔ 博斯騰湖 (白鷺洲) ➔ 庫爾勒

08:00
司機接駕，自駕向南跨越烏尉高速與天山勝利隧道，體驗地理跨越。
15:00
深度暢玩中國最大內陸淡水湖【博斯騰湖】白鷺洲景區，體驗碧水、黃沙與胡楊共生奇觀。
🏨 住宿：博斯騰賓館 / 半城百麗·梨城 / 希岸或同級【四鑽】(連宿第 1 晚)
🦢 白鷺戲水 · 沙沙共舞碧海
D3EXPLORE第 03 天
ロプ人の砂漠秘境

庫爾勒 ➔ 羅布人村寨 / 塔塔秘境 / 葫蘆島 (三選一) ➔ 庫爾勒

10:00
頂級大漠秘境三選一。推薦探訪隱匿於塔克拉瑪干沙漠邊緣的羅布人湖泊與金色水上胡楊。
🏨 住宿：博斯騰賓館 / 君潤 / 摩登四季或同級【四鑽】(連宿第 2 晚)
🛶 羅布木舟 · 浩瀚金黃大漠綠洲
D4HISTORY第 04 天
黄金の海と亀茲古城

庫爾勒 ➔ 輪台塔里木胡楊林 ➔ 庫車 (熱斯坦街 / 龜茲小巷)

13:30
深度遊覽【輪台塔里木胡楊林】，置身原始浩瀚的金黃森林海洋，看林中小火車穿梭。
18:00
進入龜茲古城庫車，漫遊熱門打卡地熱斯坦街、龜茲小巷、門巴扎，體驗異域生土雕花民居。
🏨 住宿：怡程酒店 / 庫車飯店 / 萬達悅華或同級【四鑽】
🚞 金秋輪台 · 千年大漠生命奇蹟
D5ADVENTURE第 05 天
赤い峡谷と絶景公路

庫車 ➔ 克孜利亞 / 天山神秘大峽谷 (二選一) ➔ 139秘境胡楊公路 ➔ 阿拉爾

11:30
踏足【天山神秘大峽谷】，欣賞由紅褐色巨大山體群組成的絕美赤紅深淵。
15:30
自駕穿越【139 秘境胡楊公路】，跨越三座塔河大橋，打卡九大綜合景觀與魔鬼林。
🏨 住宿：格林東方 / 蘭祥酒店 / 沙棗花香或同級【舒適】
🌋 天山奇觀 · 赤紅峽谷的壯美斧鑿
D6DESERT第 06 天
砂漠公路と夜の古都

阿拉爾 ➔ 沙漠公路 ➔ 翡翠湖 ➔ 約特干故城 ➔ 和田

09:00
自駕縱貫馳騁【塔克拉瑪干沙漠公路】，穿梭「死亡之海」茫茫黃金沙丘。
16:00
尋找黃沙深處的【翡翠湖】。隨後造訪神話般的【約特干故城】。晚間打卡和田夜市。
🏨 住宿：銀星麗呈 / 亞朵酒店 / 金鷹國際或同級【四鑽】
🐫 沙海奇蹟 · 浩瀚大漠邊緣的璀璨夜色
D7CULTURE第 07 天
異国情緒のカシュガル

和田 ➔ 澤普金胡楊 / 莎車老茶館 / 喀什古城 (三選一) ➔ 喀什

09:30
自由規劃三選一行程。特別推薦直接前往喀什古城辦理入住，深度漫遊古城小巷。
18:00
在古巷深處尋味烤鴿子、烤包子，感受熱情如火的異域歌舞氛圍。
🏨 住宿：其尼瓦克 / 美豪麗致 / 朗譽酒店或同級【四鑽】(連宿第 1 晚)
🏯 喀什老城 · 穿梭於生土與磚木的百年光影
D8PLATEAU第 08 天
パミール高原の瞳

喀什 ➔ 喀喇崑崙公路 ➔ 白沙湖全景 ➔ 木吉火山口 ➔ 喀拉庫勒湖 ➔ 塔縣

09:00
沿中巴友誼公路攀升至「世界屋脊」帕米爾高原，打卡絕美碧水銀砂白沙湖（南北雙岸）。
13:30
探秘「帕米爾之眼」【木吉火山口】，零距離觀賞休眠噴發口與連綿數百里的十八羅漢峰。
18:00
在【喀拉庫勒湖】畔遠眺「冰山之父」慕士塔格峰雪影，黃昏入住塔縣。
🏨 住宿：高原明珠 / 前海大酒店 / 喬戈里酒店或同級【舒適】(連宿第 1 晚)
❄️ 帕米爾之眼 · 白砂藍湖畔的萬年沉寂
D9ROYAL第 09 天
天空の道 · タジクの暮らし

塔縣 ➔ 紅其拉甫國門 ➔ 盤龍古道 ➔ 班迪爾藍湖 ➔ 塔吉克家訪

09:00
造訪中巴友誼公路國內段終點、世界海拔最高國界通道【紅其拉甫國門】與瓦罕走廊。
14:30
【極致重磅：盤龍古道】挑戰六百道髮夾彎公路。午後探秘「最藍之湖」【班迪爾藍湖】。
18:30
獨家高品質塔吉克族家訪：聽族群文化講解、換裝、載歌載舞，遊玩群山環繞的托格倫夏村。
🏨 住宿：高原明珠 / 前海大酒店 / 喬戈里酒店或同級【舒適】(連宿第 2 晚)
🐉 盤龍古道 · 六百道迴旋金光光軌
D10GLACIER第 10 天
氷山の王と雪柳のトンネル

塔縣 ➔ 塔合曼濕地 ➔ 樹洞公路 ➔ 慕士塔格冰川公園 ➔ 喀什

09:00
打卡【塔合曼濕地】與網紅【雪柳樹洞公路】。
12:30
攀登【慕士塔格冰川公園】(海拔 4,668 米)，近距離觀賞萬年宏偉冰舌。傍晚沿奧依塔克紅山谷返回喀什。
🏨 住宿：其尼瓦克 / 朗譽酒店 / 智涵歡朋或同級【四鑽】(連宿第 1 晚)
🏔️ 冰山之父 · 慕士塔格晶瑩剔透之美
D11WEST第 11 天
中国最西端の落日

喀什 ➔ 天山崑崙山交匯 ➔ 斯姆哈納村 (中國西極) ➔ 喀什

09:00
造訪【天山與崑崙山兩山交匯親密握手之處】，細細品味天山之駝色與崑崙山之黛色的兩大山系奇觀。
15:00
抵達神州大地太陽最晚下山、送別最後一縷陽光的西邊之村【斯姆哈納村】（中國西極），與吉爾吉斯斯坦接壤。
🏨 住宿：其尼瓦克 / 美豪麗致 / 朗譽酒店或同級【四鑽】(連宿第 2 晚)
☀️ 中國西極 · 留住太陽的最後一抹溫柔
D12DEPART第 12 天
旅立ち・また会う日まで

喀什 ➔ 出發地 (24H 送機)

全天
在酒店享用暖心早餐（需於14:00前退房），專車送往喀什機場辦理登機，滿載回憶返回溫暖的家。
🏠 住宿：溫暖的家
